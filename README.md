# MNM Virtual Services

Professional portfolio website for Mylene Martinez.

## Publish free with GitHub Pages

1. Sign in to GitHub and create a **public** repository named exactly `mnmvirtualservices.github.io`.
2. Open the repository, choose **Add file → Upload files**.
3. Upload `index.html` and the entire `assets` folder from this package. Do not upload the outer package folder.
4. Select **Commit changes**.
5. Go to **Settings → Pages**. Under Build and deployment, choose **Deploy from a branch**, then select `main` and `/ (root)`. Save.
6. After a few minutes, open https://mnmvirtualservices.github.io

## Editing later

Edit `index.html` in GitHub and commit the change. GitHub Pages will update the live website automatically.
